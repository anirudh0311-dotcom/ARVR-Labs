# Assignment 5: Marker-Based AR with Vuforia

## Overview
This repository contains the deliverables for **Assignment 5: Marker-Based AR with Vuforia**. The objective of this project is to develop an Augmented Reality (AR) application that overlays a 3D globe onto a unique, custom-designed image marker using the Vuforia Engine within Unity.

## Directory Structure
- `Marker/` - Contains the original image file (JPG/PNG) used for the custom target. *(Please make sure to add your marker image here!)*
- `Database/` - Contains the exported Vuforia Database (`Vuforia_Database.unitypackage`).
- `Screenshots/` - Contains visual evidence showing the 3D globe successfully augmented on the physical or digital marker.
- `Report.md` - The comprehensive 2-page documentation report covering the project overview and step-by-step implementation.

## Getting Started
To view the AR application:
1. Open the project in Unity.
2. Ensure the **Vuforia Engine** package is imported and configured.
3. Open the main scene.
4. Print out or display the custom marker from the `Marker/` directory on a digital screen.
5. Enter Play mode in Unity and point your webcam at the marker to see the 3D globe augmented!
